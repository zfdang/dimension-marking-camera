Installation Guide

  1. Download the APK file from the link above
  2. On your Android device, go to Settings → Security
  3. Enable Install from Unknown Sources if prompted
  4. Open the downloaded APK file
  5. Follow the installation prompts
  6. Launch DimensionCam and start annotating!

System Requirements

  • Android 7.0 (API Level 24) or higher
  • Approximately 10 MB of storage space
  • Camera permission (for taking photos)
  • Storage permission (for saving annotated images)

Build from Source

You can also build the app yourself from the source code:

git clone https://github.com/zfdang/dimension-marking-camera.git
cd dimension-marking-camera
./gradlew assembleRelease

The APK will be generated at: app/build/outputs/apk/release/
