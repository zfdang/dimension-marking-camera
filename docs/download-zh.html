安装指南

  1. 从上方链接下载 APK 文件
  2. 在您的 Android 设备上，进入 设置 → 安全
  3. 如有提示，启用 允许安装未知来源应用
  4. 打开下载的 APK 文件
  5. 按照安装提示操作
  6. 启动尺寸标注相机，开始标注！

系统要求

  • Android 7.0（API Level 24）或更高版本
  • 约 10 MB 存储空间
  • 相机权限（用于拍照）
  • 存储权限（用于保存标注图片）

从源码构建

您也可以自己从源代码构建应用：

git clone https://github.com/zfdang/dimension-marking-camera.git
cd dimension-marking-camera
./gradlew assembleRelease

APK 将生成在：app/build/outputs/apk/release/
